# 🕉️ Veda AI Lab

Modern, high-performance toolkit for AI-driven creativity and experimentation.

## 🚀 Getting Started

To launch the full laboratory, use the primary entry points in the root directory:

- **Veda AI Lab.bat**: Launches the frontend interface and local backend services.
- **VEDA_CORE.bat**: Initializes the core AI engines (Stable Diffusion, ComfyUI, Ollama).

## 📂 Project Structure

- `veda-lab-frontend/`: The primary React-based user interface.
- `server/`: Backend logic, database management, and API endpoints.
- `scripts/`:
  - `launchers/`: Auxiliary startup and setup scripts.
  - `maintenance/`: Repair and configuration tools.
  - `setup/`: Environment initialization scripts.
- `assets/`: Branding and media assets.
- `logs/`: Diagnostic and runtime logs.

## 🛠️ Requirements

- **Node.js**: For the frontend and server.
- **Python**: For AI engines (Stable Diffusion, ComfyUI).
- **Git**: For dependency management.
